Ohne I/O
========

Ohne I/O (or ``ohneio``) is an utility library to write network protocol
parsers without depending on any I/O system.

This allows the python community to then use these protocol parsers with
blocking raw sockets, asyncio, twisted, gevent, tornado, and/or
threads/multiprocesses.
